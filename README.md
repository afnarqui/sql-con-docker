# Sql con docker

Este proyecto permite correr sql desde docker

## Forma de ejecutar el proyecto en desarrollo

- instalación: `docker-compose up`



## Importante
# docker necesita una memoria mínima para ejecutar sql

````bash
  3250 megabytes of memory
````
